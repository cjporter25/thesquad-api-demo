TheSquad - API

Welcome!

    Hello! TheSquad is a personal project I undertook to explore my passion for data analysis and apply it the gaming community. The goal of this project is to develop a service tool for players of the popular online multiplayer game, League of Legends. This project focuses on providing users with interesting, fun, and useful metrics about how a specified group of players perform together as a team.

    I Obtained necessary player data by obtaining approval through the Riot Games Developer Program. Through this came the creation of a unique ID hierarchy for each Squad using Riot's provided static player identifiers, as well as a comprehensive data set that defines important metrics about each individual player and the Squad as a whole. I Developed algorithms to efficiently structure this data in an easily processable format so that retrieval and output is similar to industry standards on this topic. Lastly, I created and deployed a RESTful API (what you see below!) using Flask and Heroku cloud hosting services.

    The purpose of this project was to offload the collection, storage, and management of this data away from a theoretical "TheSquad" website. It's hard to say if that will ever come to fruition since I'm working on this alone, however, I found it useful in helping narrow the scope and purpose of the project thus far. I plan to analyze the stored data and utilize these results to develop a tailored set of metrics that represent a team's relative performance under variety of conditions. Acquiring basic stats is the first step. Once secured, interpreting what this data means, in the scope of the game, can provide valuable insight into a team's perfomance together rather than individual players.
