Match - "In-Game Match Data"

  • [GET] www.thesquad-api.com/get-match-history/<riot_id>/?tagline=<string>&count=<string>

    Every match in League of Legends has a unique match ID. This request grabs the recent match history of a specified player and returns a JSON array of match ID's. This JSON array can be used to fetch detailed information about each match using another API call that takes a match ID as a parameter.
    Input & Query Parameters
    NAME VALUE DATA TYPE DESCRIPTION
    riot_id "player1" string This is simply the riot ID of a given player without their tagline. The ID is alone in this case due to differing region taglines.
    tagline= "NA1" or "1234" string This is a player's tagline that is attached to their RiotID. Default = "NA1"
    count= "20" or "90" string (number) This is the requested count for how many recent matches to pull. Range: 1-90 recent matches. Default = "20"
    Response Codes (NIY = Not Yet Implemented)
    HTTP STATUS CODE DESCRIPTION
    200 SUCCESS
    400 BAD REQUEST - Invalid parameter or incorrect URL syntax
    403 (NYI) FORBIDDEN - Invalid API key, incorrect path
    404 NOT FOUND - URL syntax was okay. Provided riot_id & tagline were not valid
    Response Output
    NAME DATA TYPE DESCRIPTION
    match_history JSON Array [string, string, ...] JSON Response containing an array of match ID strings. The number of items is equal to the parameter "count". Default is 20 ID's.
