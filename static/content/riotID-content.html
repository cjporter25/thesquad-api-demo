RiotID - "Player Data"

[GET] www.thesquad-api.com/get-player-info/<summonerName>

  • Input or Query Parameters
    NAME VALUE DATA TYPE DESCRIPTION
    summonerName "Chrispychickn25" string When querying for a player by their summoner name, their in-game name is required as part of the URL.
    Response Codes (NIY = Not Yet Implemented)
    HTTP STATUS CODE DESCRIPTION
    200 Success
    400 (NYI) Bad Request - Invalid parameter, incorrect syntax
    404 (NYI) Not Found - SummonerName doesn't exist
    Response Output
    NAME DATA TYPE DESCRIPTION
    acctID string The Account ID, a permanent unique string, connected to the provided summonerName
    id string The ID created by riot, presumably for their servers. This ID is not pertinent for other functions of the API.
    lvl long The current account level of the provided summonerName
    puuID string The puuID, is another permanent unique string, connected to the provided summonerName. Used extensively in gathering match data and overall database organization.
