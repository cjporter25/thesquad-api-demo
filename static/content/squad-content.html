Squad - "Team Data"

[GET] www.thesquad-api.com/retrieve-squad-data/?<p0>&<p1>&<p2>&<p3>&<p4>

  • Input or Query Parameters
    NAME VALUE DATA TYPE DESCRIPTION
    p0 Ex: "player1#NA1" string The first squad member's riotID. Must contain valid riotID, followed by the player's tagLine. This applies to all subsequent riotID's.
    p1 "player2#NA1" string The second squad member's riotID.
    p2 "player3#NA1" string The third squad member's riotID.
    p3 "player4#NA1" string The fourth squad member's riotID.
    p4 "player5#NA1" string The fifth squad member's riotID.
    Response Codes (NIY = Not Yet Implemented)
    HTTP STATUS CODE DESCRIPTION
    200 SUCCESS
    400 BAD REQUEST - Invalid parameter, incorrect syntax, too few members provided
    403 (NYI) FORBIDDEN - Invalid API key, incorrect path
    404 (NYI) NOT FOUND - One or more of the riotID's doesn't exist
