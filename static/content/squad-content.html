Squad - "Team Data"

[GET] www.thesquad-api.com/retrieve-squad-data/?<p0>&<p1>&<p2>&<p3>&<p4>

  • Input or Query Parameters
    NAME VALUE DATA TYPE DESCRIPTION
    p0 Ex: "player1#NA1" string The first squad member's riotID. Must contain valid riotID, followed by the player's tagLine. This applies to all subsequent riotID's.
    p1 "player2#NA1" string The second squad member's riotID + tagline.
    p2 "player3#NA1" string The third squad member's riotID + tagline.
    p3 "player4#NA1" string The fourth squad member's riotID + tagline.
    p4 "player5#NA1" string The fifth squad member's riotID + tagline.
    Response Codes (NIY = Not Yet Implemented)
    HTTP STATUS CODE DESCRIPTION
    200 SUCCESS
    400 BAD REQUEST - Invalid parameter, incorrect syntax, too few members provided
    403 (NYI) FORBIDDEN - Invalid API key, incorrect path
    404 (NYI) NOT FOUND - One or more of the riotID's doesn't exist
    Response Output
    NAME RETURN DATA TYPE DESCRIPTION
    ARAM_sqMatchesLost Integer Total number of matches recorded where this squad LOST. [MatchesLost] + [MatchesWon] = [MatchesPlayed]
    ARAM_sqMatchesPlayed Integer Total number of matches played & recorded where this squad was present
    ARAM_sqMatchesWon Integer Total number of matches recorded where this squad WON. [MatchesWon] + [MatchesLost] = [MatchesPlayed]
    ARAM_sqWinrate Double Winrate of this squad for the game mode: ARAM. Calculated by dividing [MatchesWon]/[MatchesPlayed]
    ARAM_sqWinrates_Assasin List [[String, Double, Integer], [], []...] A list of each squad member(string), their winrate, and the number of games played as an Assasin. Ordered by highest winrate. Internally ordered by name for clarity
    ARAM_sqWinrates_Enchanter List [[String, Double, Integer], [], []...] Exact same structure, ordered by highest winrate for games played as an Enchanter.
    ARAM_sqWinrates_Fighter List [[String, Double, Integer], [], []...] Exact same structure, ordered by highest winrate for games played as a Fighter.
    ARAM_sqWinrates_Mage List [[String, Double, Integer], [], []...] Exact same structure, ordered by highest winrate for games played as a Mage.
    ARAM_sqWinrates_Marksman List [[String, Double, Integer], [], []...] Exact same structure, ordered by highest winrate for games played as a Marksman.
    ARAM_sqWinrates_Support List [[String, Double, Integer], [], []...] Exact same structure, ordered by highest winrate for games played as a Support. This term is reserved for champions not traditional supports playing the support role. Since all viewable return values are based of expected role, this stat can be ignored.
    ARAM_sqWinrates_Tank List [[String, Double, Integer], [], []...] Exact same structure, ordered by highest winrate for games played as a Tank.
